# Craftable Jute
This simple mod allows you to craft red and blue jute at the artisan table.

## Recipes
3 Linen Thread + 1 Raspberry = 1 Red Jute

3 Linen Thread + 1 Magecap = 1 Blue Jute

Both recipes are crafted at the Artisan Table.
